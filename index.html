ShipMaster

ShipMaster was a collaberative project that I worked on with some folks that were in my Bootcamp and it was developed to help users manage their package shipments. ShipMaster also enables users to see which carrier will be shipping their package (unfortunatley, not carriers are recognized). Utilizing ReactStrap allowed us to design a UI that would benifit the user's ability to navigate the site. User's are able to create accounts by entering their email and password. Each search is done by entering a tracking number and the user will be given information on their shipment.
